Sensori.js è il file che gestisce gli input e gli output agendo sui pin del rasberry pi

Avviare il programma
1)Spostarsi all'interno della cartella node-dht-sensors
2)Aprire un terminale all'interno della stessa cartella
3)Digitare node Sensori.js

Assicurarsi che i pin all'interno del programma corrispondano a quelli effetivamente utilizzati
sulla scheda

Se si utilizza una scheda inferiore alla versione 3 è necessario dotarla di apposito cavo con
uscita rj-45



All'interno della cartella serverhome sono presenti le pagine html visualizzabili dall'utente,
come la scelta della propria smarthome e il proprio profilo.
