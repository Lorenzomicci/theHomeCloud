---------------------------------------------------theHomeCloud--------------------------------------------------------------

File di ingresso: server.js

Avvio

Su piattaforma Unix

1) Aprire li terminale e digitare la stringa: sudo apt-get install node
2) Spostarsi all'interno di questa cartella trmite gli opportuni comandi cd
3) Avviare il server digitando da terminale node server.js

Su piattaforma Windows
1)Scaricare nodejs dal sito https://nodejs.org/en/
2)Aprire il prompt dei comandi
3)Spostarsi all'interno della cartella di installazione di node   
4)digitare node *percorso de file server.js*.js
